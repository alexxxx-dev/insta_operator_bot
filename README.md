# insta_operator_bot

Простое окно для работы с Instagram Direct.

Что есть в проекте:
- подключение аккаунта по сохранённой сессии;
- просмотр диалогов;
- чтение сообщений;
- отправка ответов из интерфейса.

Запуск:

```bash
pip install instagrapi
python gui.py
```

Перед запуском проверь `config_operator.json` и путь к сессии.
